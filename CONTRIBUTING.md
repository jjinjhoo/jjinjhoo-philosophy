# Contributing Guide

환영합니다! 이 프로젝트는 오픈소스 철학책입니다.

## PR 범주
- PATCH: 오탈자·문장 다듬기
- MINOR: 사례·도표·절 추가, 해설 보강
- MAJOR: 핵심 정의·구조 변경 (이 경우 Issue로 먼저 제안)

## 스타일 & 원칙
- 명료함과 울림의 균형
- 핵심 축을 흔들지 않기: (1) 주(主), (2) 선택×실행, (3) 돈=에너지, (4) 하나의 전체
- 인용/이미지는 출처 명시

## 워크플로
1. Issue로 제안
2. 브랜치 생성 → 수정 → PR
3. 토론 & 리뷰 → 머지
